Introduction

This video walks you through step-by-step on how to share your Windows Phone 8 Screen over the Internet. The only requirements are that you are using a Nokia Device and the Nokia Black update is installed.

Steps

  1. Check to see if your device already has the Nokia Black update. It has been rolling out in devices since January 2014.
    For more info on everything that is included please see: http://www.nokia.com/global/windows-phone-black-update/
  2. If you don’t have it then you can download the “Preview for Developers” app and install it on a list of devices located below. Please see the notes on who can install the update . 
  3. Install and Run the “Nokia Beamer” app on your phone.
  4. Optionally go to https://beam.nokia.com/ if you are sharing locally. Otherwise send out a link to the URL.

Other Notes:

The current Nokia phones are supported at the time of this recording: Nokia Lumia 520 520T 521 525 526 620 625 720 720T 810 820 822 920 920T 925 925T 928 Icon 1020 1320 and 1520.

In order to install the update from the “Preview for Developers” app you must meet one of the following three requirements: (a) have an active Windows Phone Dev Center account; (b) have an active App Studio account; OR (c) be running a registered developer device (‘dev unlocked’ phone)
