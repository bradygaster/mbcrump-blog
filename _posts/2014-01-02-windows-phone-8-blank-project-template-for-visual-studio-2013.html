Introduction

A lot of people were happy with the blank Windows Phone 8 Project Template that I created for Visual Studio 2012. Over the holidays I had a request to do the same thing for Visual Studio 2013. Since Visual Studio 2013 is backwards compatible with Visual Studio 2012 you can actually use the same project template as before.

Installation

The template is available here to download. You will need to copy the zip file to the following directory (creating only the Windows Phone directory) C:\Users\<your user name>\Documents\Visual Studio 2013\Templates\ProjectTemplates\Visual C#\Windows Phone\

Note: Only Visual C# is supported.

After the zip is copied inside the folder launch Visual Studio 2013 and navigate to your Windows Phone templates and select “Windows Phone App – Blank” as shown below.
